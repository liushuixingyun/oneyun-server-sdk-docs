## 引用
在使用 `IVR` 之前，开发者应[安装](../index.md#安装)并“引用” Oneyne 的程序包。
引用部分的代码是这样的：

{% sample lang="js" -%}
```js
let oneyun = require('oneyun');
```

{% sample lang="js" -%}
Oneyun SDK 的根名称空间是 `oneyun`

{% sample lang="php" -%}
```php
<?
require "vender\autoload.php";
```

{% sample lang="js" -%}
Oneyun SDK 的根名称空间是 `Oneyun`
